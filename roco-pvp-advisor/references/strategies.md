# 洛克 PVP 通用策略原则

## 一、联防方法论

联防是洛克 PVP 最重要的防御性操作——通过换精灵让对手的攻击打在抗性精灵身上。

### 什么时候换

- 对手当前精灵克制我方 → 换抗性精灵上来吃伤害
- 对手正在强化而我方无法打断 → 换拦截手上场压制
- 对手能量即将耗尽 → 换我方需要发育的精灵上来安全开状态
- 我方当前精灵已经完成任务（开了天气/叠了印记）→ 换下一个

### 换谁的代价最小

- 优先级 1：血量厚的联防位
- 优先级 2：属性抵抗对手技能
- 优先级 3：本身定位为"可以牺牲"的探路精灵
- 绝不要换：后期收割核心、体系发动机

### 联防的收益计算

```
我方换精灵 → 对手攻击命中
        → 我方损失：若干血量
        → 对手损失：高能耗技能的能量
        → 我方获得：抗性精灵的场位优势

如果对手用 5 费技能打在我方抗性精灵上只掉了 20% 血
→ 对手亏了 5 能量，我方只亏了 20% 血
→ 这是稳赚的交换
```

### 联防的风险

- 对手预判你要换 → 他不开大招，改用状态回能 → 你白换，还亏了场位
- 对手预判你要换 → 他换克制你联防位的精灵 → 你换上来反而被压制

## 二、拦截手价值

拦截手不是用来赢的，是用来不让对手赢的。

### 拦截手的特征

- 带有打断类技能（如有效预防、截拳、地刺）
- 带有应对攻击后反制能力（如防御反击、听桥）
- 属性或特性克制主流强化核心
- 速度不一定要快，但出场时机要精准

### 拦截手的操作时机

- 对手强化精灵刚完成一次强化 → 立刻换拦截手上场
- 对手能量不足以连续出大招 → 拦截手可以安全发挥
- 拦截手本身血量充足 → 可以硬吃一次伤害然后反制

### 经典拦截手举例

| 精灵 | 拦截方式 |
|------|---------|
| 锤头鹳 | 迅捷水弹枪（2费魔攻水系）打火系和巨鼓象，5次博弈空间 |
| 海豹船长 | 听桥+防御反击叠被动，水刃收割 |
| 泥吼牙 | 特性「无差别过滤」锁连击数=2，落石秒火系，单防火队 |
| 魔力猫（喵喵） | 草系抵抗地面，移花接木反弹物理，单防地武整队 |
| 翠顶夫人 | 飞羽清增益印记，冰爪打龙鱼 |
| 卡瓦重 | 冰+草三倍抗翼王，280速速冻控速 |

## 三、赔率思维

每一步都不是单看"这一下打多少"，而是看"如果我猜错，后果多严重"。

### 后悔值最小化

```
对手可能的动作：
A: 攻击（概率 40%）
B: 防御（概率 35%）
C: 状态（概率 25%）

我方候选：
候选 1（攻击）：打中 A 赚 / 碰 B 亏 / 碰 C 赚
候选 2（防御）：碰 A 大赚 / 碰 B 空过 / 碰 C 亏
候选 3（换精灵）：碰 A 亏血 / 碰 B 赚节奏 / 碰 C 赚节奏

选择后悔值最低的——就是即使猜错，也不会崩盘的动作。
```

### 常见决策模板

- **对手能量充足 + 有高费大招** → 倾向防御（他大概率打）
- **对手能量很低** → 倾向攻击或换精灵（他大概率聚能或换）
- **对手状态技能收益极高（如强化/天气）** → 倾向打断（攻击+应对状态技能）
- **对手已经有速度优势** → 优先考虑防御或换精灵（不能硬拼速度）
- **对手阵容后期有收割核心** → 倾向保存魔力和拦截手，不要贪
- **我需要体系启动** → 先用小技能试探，找安全回合开状态

## 四、能量博弈

能量是洛克 PVP 的"节奏货币"。

### 能量压制

- 如果对手能量 ≤ 3 → 他几乎无法出高费技能 → 你的攻击窗口
- 如果对手能量 ≥ 7 → 他可以出任何技能 → 你要小心
- 如果对手空能量 → 他只能聚能或换精灵 → 你可以安全发育或压制

### 能量管理原则

- 保持 ≥ 4 能量，留出应对空间
- 不要连续用高费技能——用一次后容易被对手预判防御
- 状态回能技能是节奏的关键——它让你在不换人的情况下维持能量

## 五、体系识别与打断

### 如何识别对手体系

看队伍构成：
- 水系 ≥ 2 只 + 天气手 → 雨天队
- 多只火系 + 燃薪虫/火焰猿 → 火队/平衡火
- 冰系为主 + 雪影娃娃 → 雪天队
- 海豹船长 + 海枝枝 + 音速犬 + 恶魔狼 → 借用狼王（看前场就知道）
- 帅帅魔偶 + 海豹船长 + 棋绮后 → 地武队
- 怖哭菇 + 化蝶 + 龙息帕尔 → 星陨队
- 千棘盔/裘卡 + 厉毒修罗 → 毒队
- 虫系为主 + 花衣蝶 → 虫队
- 机幕方舟 → 过山车/积木方舟
- 布莱克岩 + 食尘短绒 + 地系为主 → 沙暴队
- 无明显特征 → 平衡联防队

### 打断原则

- 优先针对体系发动机（天气手、强化核心）
- 在对手启动的"中间回合"打断——不是在启动前太早，也不是启动后太晚
- 打断的手段：应对技能、换拦截手、用攻击+应对状态技能威慑

## 六、魔力保护

- 每只精灵都是"一条命"——你只有 4 条。
- 残血精灵 ≠ 没用的精灵。只要它还活着，就可以联防、吃技能、开天气、叠印记。
- **不要着急送残血精灵**——留着它当炮灰可能救核心精灵一命。
- 优势局：保守打，保护魔力。
- 劣势局：积极主动，找翻盘点。

---

## 七、信息差利用

虽然双方队伍可见，但仍有信息差：

### 对手不知道的
- 我方精灵的具体技能携带（冷门技能出奇制胜）
- 我方精灵的性格和速度线
- 我方精灵的血脉（愿力冲击出奇兵）

### 我方需要探明的
- 对手每只精灵已暴露了什么技能？
- 对手的能量消耗习惯（喜欢打还是喜欢防？）
- 对手的联防链（谁换进来扛？）

### 利用方式
- 前几回合用小能耗技能试探对手操作习惯
- 标记对手已暴露的技能，推断它可能没带什么
- 故意露出破绽诱导对手出特定动作，然后反制

## 八、常见误区

1. **"我有克制技能，打就行"** — 对手知道你有克制，可能防御或换人，你打空就亏。
2. **"我血多，没事"** — 血多但能量耗尽 = 接下来几回合只能挨打或换人。
3. **"我要把这只精灵的所有 Buff 用完再换"** — Buff 离场清空，但有价值的 Buff 可以传给下一只（某些精灵特性支持）。
4. **"对面残血我先收掉"** — 残血精灵不一定优先处理。有时候留着残血反而限制对手换人（他不敢安全换出核心）。
5. **"这精灵死了就输了"** — 不一定。你有 4 点魔力。失去关键精灵可能只是"这条路走不通了"，转另一条胜利路线。

## 九、S2 环境特有策略

### 针对巨鼓象
- 巨鼓象物防极高但魔防一般 → 优先用魔攻手（海枝枝、贝古斯）逼退
- 特性「合拍」双方技能系别/类型/能耗相同就叠攻防 → 不和它站场对位
- 引燃（灼烧）可逼退但会让它叠特性 → 最好在它齿轮扭矩叠起来前解决

### 针对霹雳迪迪
- 迪迪130速+139魔攻，落雷轮转叠到300+伤害一发一个
- 用草系（魔力猫、卡瓦重）稳定拦截
- 用愿力水灵天洪赌放晴→一发秒杀
- 用锤头鹳水弹枪2费压血

### 针对借用狼王
- 在前场阻止狼王队友送死节奏（不要让4只轻松送掉）
- 用迪迪首发压狼王队友血线
- 化蝶高魔攻站场→狼王吸不过
- 骨龙复活会让狼王被动倒扣

### 针对雪天队
- 带洗礼技能清除冻结
- 用冰钻布鲁斯（冰系叛徒）
- 愿力抓雪影娃娃阻止开冬至
- 不要和雪天队拖时间——冻结越叠越深

### S2 能量管理特别提醒
- 锤头鹳的2费迅捷水弹枪是最有效的低费高质量压制
- 冰封bug导致雪天队可以无限能量——遇到必须打断
- 龙鱼龙吟后buff增幅极强——一旦蓄力成功就是巨大优势
