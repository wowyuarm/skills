# 洛克王国世界 PVP 完整机制

## 基础资源

### 魔力值
- 双方各 4 点魔力。
- 己方每有一只精灵血量归零 → 扣 1 点魔力。
- **魔力值先归零的一方判负。**
- 魔力值不可恢复。
- 战术含义：卖精灵（主动送掉一只）是合法策略，但要精确计算"还剩几条命"。连续被杀两只就直接输掉比赛。

### 能量值
- 上限 10，使用技能消耗左侧标注的能耗。
- 能量不足 → 无法使用该技能。
- 能量归零 → 可选动作只剩：换精灵，或使用默认技能"聚能"。
- "聚能"：消耗当前回合，获得 5 能量。不占用正常技能位。
- 部分攻击/状态技能附带回复能量效果。
- **让对手空能量比打掉对手血量往往更重要**——空能量 = 丧失行动选择权。

### 生命值
- 伤害公式：(己方攻击 - 敌方防御) × 技能倍率 × 属性克制系数（0.5 / 1 / 2）。
- 没有暴击、没有命中率、没有闪避。
- 技能 UI 会实时显示伤害预期颜色：红（克制）/ 白（普通）/ 绿（抵抗）。

## 动作类型

### 攻击（物攻 / 魔攻）
- 直接造成伤害。
- 部分攻击技能带有额外效果（吸血、连击、能耗变化、印记叠加等）。
- 某些攻击技能有"应对"条件：如果对手正在使用某类技能，触发额外效果。
  - 例：偷袭"应对状态：本次技能威力变为 3 倍"——如果对手这回合用状态技能，伤害 x3。

### 防御
- 本回合获得高额减伤（通常 70%-90%）。
- **防御技能不能连续使用**（防止无限防御）。
- 大部分防御技能带有"应对攻击/状态/防御"条件：
  - 应对成功 → 触发额外奖励（回能、反击、增益、打断等）。
  - 应对失败 → 只获得基础减伤，亏了一回合。
- 有些防御技能应对成功后效果极强：
  - 例：防御反击"减伤 80%，应对攻击：全技能威力+40"
  - 例：听桥"减伤，应对攻击：以对手攻击伤害反击，可能直接反杀"

### 状态
- 不直接造成伤害。用于 buff 自身、debuff 敌方、回复、叠加印记、改变天气等。
- 部分状态技能有"应对防御/攻击"条件。
  - 例："闪燃"下一次攻击技能威力翻倍，应对防御：改为威力变为 4 倍"
- 状态技能是体系启动的关键：强化精灵、开天气、叠印记都在这里完成。

### 换精灵
- **优先级最高**，不受速度影响，总是先于攻击/防御/状态执行。
- 换上场的新精灵承受对手本回合的攻击。
- 常用场景：
  - 联防：换抗性精灵吃伤害，用血量换对手能量。
  - 找优势对位：换克制精灵压制对手。
  - 躲状态：避免被对手的 debuff 或印记命中。
  - 启动体系：换天气手或强化精灵上场。

## 应对系统

洛克 PVP 的核心博弈机制。类似猜拳，攻击 / 防御 / 状态三者互有克制关系：

- **攻击克制状态**：攻击技能打中对手状态技能 → 很多攻击技能会触发额外效果（威力翻倍、打断等）。
- **防御克制攻击**：防御技能挡住对手攻击 → 触发防御技能的应对奖励（回能、反击、增伤等）。
- **状态克制防御**：对手防御时你开状态 → 你安全发育，对手空过一回合。

**关键认知**：
- "猜中" → 收益巨大，可能扭转战局。
- "猜错" → 亏损（攻击打防御 → 伤害大减，状态被打断 → 浪费回合）。
- 因此你不是在选"伤害最高的"，而是在选"后悔值最低的"。

**高级应对**：
- 某些攻击技能有"应对状态"条件——如果对手开状态，威力 x3。这在对手需要强化时是巨大的威慑。
- 某些防御技能有"应对攻击"条件——对手打你反而触发反杀。
- 换精灵可以同时规避所有应对——但换人也意味着主动卖血或被对手预判压制。

## 速度与先手

- 每回合，速度高的一方先行动（攻击/防御/状态）。
- 换精灵不受速度影响，必定先于任何技能执行。
- 部分技能有"先手"效果（如先手+1），可以无视速度抢先行动。
- 部分技能可以改变速度（增益/减益）。
- **战斗中点击精灵头像可以查看双方速度对比**。

## Buff 与印记

### Buff（增益/减益）
- 显示在精灵头像下方的层数图标，每层 ±10%。
- 作用于当前在场精灵。
- **离场时清空**。换精灵 = 丢 Buff。
- 常见：物攻+、魔攻+、物防+、魔防+、速度+。
- debuff：
  - 灼烧：持续掉血，回合结束会衰减
  - 中毒：持续掉血且回合间保留中毒效果，离场清空

### 印记
- 印记是高级 Buff，**不会因换精灵而清空**，全局生效。
- 正面印记示例：
  - 湿润印记：减我方水系技能能耗
  - 光合印记：回合结束回能，每层印记回复一点能量
  - 蓄势印记：增加技能威力，但能耗增加
- 负面印记示例：
  - 星陨印记：非幻系技能会引爆印记，层数越多伤害越高
  - 萌化印记：叠加后触发特殊效果
- 印记是体系队的重要基础：通过前期叠印记，后期爆发。

## 属性与克制

- 18 种属性，克制关系简化：克制系数 2 / 普通 1 / 抵抗 0.5。
- 精灵可拥有主+副双属性。技能属性独立于精灵属性。
- 双属性克制叠加：若技能同时克制主副属性，系数相乘。
- **属性除克制外，每种属性有独特的技能风格**：
  - 火系：频繁叠加攻击
  - 水系：降低能耗
  - 机械系：技能位"传动"
  - 虫系：获得"奉献"buff 为全队增益
  - 武系：从应对中获得大量增伤

## 天气系统

部分精灵可以召唤天气，持续 N 回合，全场生效。

### 雨天
- 效果：水系技能增伤75%。
- 常见天气手：水灵（落雨技能）。
- 战术意义：水系精灵可以频繁使用高费技能，形成压制。

### 沙暴
- 效果：地系技能能耗减半。
- 常见天气手：布莱克岩、食尘短绒（沙涌技能）。
- 战术意义：地系精灵的耐久+输出双重受益。

### 暴风雪
- 效果：每回合结束时，对方场上精灵获得2层冻结（无需冰系技能命中）。
- 常见天气手：雪影娃娃（冬至技能，7费，持续8回合）。
- 冻结层数：1层=每回合5%最大HP伤害。叠加源：冬至、冰墙、雪影入场、雪怪被动、冰点、焚烧烙印等。
- 战术意义：控制+冻结消耗+雪熊引爆星陨印记斩杀。
- **注意**：冰封技能（4费95威力冰系）存在社区广泛认定的bug——技能文本"敌方没有一层冻结，回复一个能量"，但实际判定检查对方全体精灵（含背包）的冻结层数，而非仅前台。同文本的"碎冰冰"不查后备。官方S2赛季未修、未回应。

### 通用提醒
- 天气改变后覆盖前一个天气，不会叠加。
- 天气手一旦被击杀，天气可能无人延续。
- 队伍里通常需要 2 个天气手以防万一。
- **谁先开天气谁吃亏**：后手开天气可以顶掉对方的天气。

## 传动机制（S2 机械系核心）

部分机械系技能带"传动 X"属性：

- "传动 X"：回合开始时，该技能在技能栏中向下移动 X 个位置。
- 精灵特性往往与技能位置相关（如"1 号位技能威力+XX""3 号位技能能耗-X"）。
- 贝古斯特性「贪心算法」：1号位技能传动后自动挂6层灼烧，配合「燃尽」「磁爆」等位置敏感技能。

**机幕方舟专属技能「过山车」**（8费状态技能）：
- 使我方**所有精灵**的技能跨精灵向下移动一个位置（1号精灵4号位→2号精灵1号位，依此类推）
- 已死亡精灵也参与传动（送掉也能传）
- 派对精灵（789号位）同样参与传动
- 帕帕斯卡特性可使传过来的技能获得迅捷
- 方舟自身受克制伤害+25%（火/水/武系威胁大）

## 血脉系统

- 精灵除了主副属性外，还有一个"血脉"属性。
- 血脉不影响属性克制关系。
- 血脉的主要作用：
  - 学习其他属性的特定技能
  - 满足特定进化条件
  - 使用特定血脉的愿力冲击（类似宝可梦的Z招式）
- 可以通过道具修改血脉。

## 愿力冲击

- 背包道具技能，一局可使用次数有限。
- 效果：使精灵的首个技能在本回合变为隐藏血脉属性的攻击。
- 若应对到对手状态技能 → 威力 ×1.5。
- 战术含义：逆属性反杀的关键手段。

## 首领进化

- 特定精灵可携带首领信物，在战斗中"超进化"。
- 进化后获得新特性、更高数值。
- 战术含义：类似于 Mega 进化，是重要的战力跃升点。

## PVP 对局节奏

### 试探阶段（前 1-2 分钟）
- 根据敌方首发推测体系类型。
- 判断是否需要调整首发策略。
- 用小能耗技能试探对手的操作习惯。

### 博弈阶段（主要阶段）
- 双方互相破坏对方套路。
- 通过应对、联防、换宠拉扯。
- 追求以最小代价化解对手威胁。

### 处决阶段（末尾）
- 一方资源耗尽或核心被击杀 → 投降或被推队。
- 仍有翻盘可能的选手坚持到最后一刻。

## 伤害区间判断

LLM 不适合精确伤害计算。在缺少计算器时，使用区间判断：

| 区间 | 含义 |
|------|------|
| 能秒 | 当前血量下确定一击击杀 |
| 大概率秒 | 属性克制+高威力技能，大概率击杀 |
| 二段可杀 | 需要两回合才能击杀 |
| 打残 | 造成大量伤害但不足以击杀 |
| 蹭血 | 伤害较低，主要用于叠印记或消耗 |
| 打不动 | 属性抵抗+防御高，几乎没伤害 |
